To push to GitHub as Shivam-0803, you need to create a Personal Access Token.

Follow these steps:

1. Log in to GitHub.com as Shivam-0803
2. Go to Settings > Developer settings > Personal access tokens > Tokens (classic)
3. Click "Generate new token" and select "Generate new token (classic)"
4. Give it a descriptive name like "Mini CRM Push Token"
5. Select the "repo" scope
6. Click "Generate token" and copy the token
7. In your terminal, run these commands (replace YOUR_TOKEN with the token you copied):

   git remote set-url origin https://Shivam-0803:YOUR_TOKEN@github.com/Shivam-0803/mini-crm-.git
   git push -u origin main

IMPORTANT: Do not share this token with anyone - it provides access to your GitHub account!
